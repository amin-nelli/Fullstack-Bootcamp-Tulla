# M1-TullaShop

Projeto desenvolvido como atividade do Módulo 1 do Bootcamp Full Stack da Tulla.

## Descrição

Este projeto foi criado com o objetivo de praticar os fundamentos de HTML e CSS, trabalhando estrutura de páginas, organização visual e estilização de elementos.

## Tecnologias utilizadas

- HTML5
- CSS3

## Objetivos do projeto

- Praticar a estruturação de páginas com HTML
- Aplicar estilização com CSS
- Organizar elementos visuais de forma clara
- Desenvolver a base inicial em front-end

## Aprendizados

Durante o desenvolvimento deste projeto, pratiquei:

- Estrutura semântica com HTML
- Organização de conteúdo em página web
- Estilização com CSS
- Ajustes visuais de layout
- Boas práticas iniciais de desenvolvimento front-end

## Como visualizar

1. Baixe ou clone este repositório
2. Acesse a pasta `M1-TullaShop`
3. Abra o arquivo `index.html` no navegador

Observação: Os botões clicáveis são a logo da TullaShop, o menu Vendas TullaShop e o Carrinho.
